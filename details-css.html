البرمجة بلغة CSS

تاريخ: 13 يوليو 2025

CSS هي لغة مسؤولة عن تصميم صفحات الويب، وتستخدم للتحكم في الألوان، والخطوط، وتوزيع العناصر.

محتوى المقال بالتفصيل:

  • الغرض: فصل المظهر عن المحتوى.
  • كيفية العمل: استخدام قواعد لتحديد الخصائص.
  • الخصائص: مثل اللون والخطوط والتباعد.
  • القيم: مثل red و16px وغيرها.
  • أنواع CSS:
    • الأنماط المباشرة (Inline)
    • الأنماط الداخلية (Internal)
    • الأنماط الخارجية (External)
