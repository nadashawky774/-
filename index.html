البرمجه بلغة html

تاريخ:14 يوليو 2025

HTML، أو لغة ترميز النص التشعبي، هي اللغة الأساسية المستخدمة لبناء صفحات الويب. تُستخدم لإنشاء هيكل ومحتوى صفحات الويب،
مما يسمح للمتصفحات بعرض النصوص والصور والروابط والعناصر الأخرى بشكل منظم. تعتمد HTML على مجموعة من "العناصر" التي تحدد وظيفة كل جزء من المحتوى،
مثل الفقرات والعناوين والقوائم والجداول.

شرح مبسط

  • الهدف

    • إنشاء صفحات الويب وتحديد هيكلها ومحتواها.
  • العناصر:

    • HTML يعتمد على "العناصر" (مثل

    للفقرات و

    للعنوان الرئيسي) لتحديد وظيفة كل جزء من المحتوى.
  • الوسوم:

    • كل عنصر محاط بوسوم (علامات) تبدأ بعلامة < وتنتهي بعلامة >.
  • التركيب:

    • يتم تنظيم العناصر داخل بعضها البعض لتشكيل شجرة هيكلية للمستند.
  • التكامل مع التقنيات الأخرى:

    • HTML غالبًا ما يُستخدم مع CSS لتنسيق المظهر، و JavaScript لتفعيل التفاعلية في الصفحات.

البرمجه بلغةcss

تاريخ13 يوليو 2025

CSS هي لغة تصميم تستخدم لتحديد شكل وعرض مستندات HTML (أو XML). باختصار، CSS هي المسؤولة عن جعل صفحات الويب تبدو جميلة وجذابة من خلال التحكم في الألوان، والخطوط، والتخطيط، وغيرها من جوانب العرض.


شرح CSS:


  • الغرض:

    • تهدف CSS إلى فصل تصميم صفحة الويب (المظهر) عن محتواها (HTML). هذا يجعل من السهل تحديث وتعديل مظهر الموقع دون الحاجة إلى تغيير هيكل المحتوى.

  • كيفية العمل:

    • يتم تطبيق CSS على عناصر HTML باستخدام "قواعد". تتكون كل قاعدة من محدد (selector) يحدد العنصر (أو العناصر) المراد تنسيقها، و "كتلة إعلان" (declaration block) تحتوي على خصائص وقيم.

  • الخصائص:

    • تحدد جوانب التنسيق التي سيتم تطبيقها على العنصر. تشمل الخصائص الشائعة color (اللون)، وfont-family (نوع الخط)، وbackground-color (لون الخلفية)، وmargin (الهامش)، وpadding (الحشو)، وغيرها.

  • القيم:

    • تحدد القيمة التي سيتم تعيينها للخاصية. على سبيل المثال، يمكن أن تكون قيمة color هي red أو #FF0000 (كود اللون الأحمر)، ويمكن أن تكون قيمة font-size هي 16px (16 بكسل).القيم:


    أنواع CSS

    :
  • الأنماط الداخلية (Inline Styles): يتم تعريف الأنماط مباشرة داخل عنصر HTML باستخدام السمة style.


  • الأنماط الداخلية (Internal Styles): يتم تعريف الأنماط داخل قسم في قسم من مستند HTML.


  • الأنماط الخارجية (External Styles): يتم تعريف الأنماط في ملف CSS منفصل، ثم يتم ربطه بمستند HTML باستخدام الوسم .

البرمجه بلغة JavaScript

تاريخ12 يوليو 2025

جافا سكريبت (JavaScript) هي لغة برمجة تُستخدم بشكل أساسي لجعل صفحات الويب تفاعلية. هي لغة برمجة عالية المستوى وديناميكية تُفسّر (بدلاً من أن تُبنى) وتعمل بشكل رئيسي في المتصفحات، ولكنها تستخدم أيضًا في تطبيقات أخرى مثل Node.js.


شرح مبسط:


  • التفاعلية:

    • جافا سكريبت تسمح للمواقع الإلكترونية بالتفاعل مع المستخدمين. فبدلاً من مجرد عرض معلومات ثابتة، يمكن لجافا سكريبت تحديث الصفحة، والتعامل مع المدخلات، وإجراء العمليات الحسابية، وكل ذلك بشكل حي ومباشر على صفحة الويب.


  • أجزاء أساسية:

    • جافا سكريبت هي جزء من ثلاثية أساسية لتطوير الويب، إلى جانب HTML (لتحديد بنية الصفحة) و CSS (لتصميمها).


  • العمل في المتصفح:
    • عند زيارة موقع ويب، يقوم المتصفح بتحميل كود جافا سكريبت وتنفيذه، مما يجعله يتفاعل مع الصفحة.


  • أمثلة على الاستخدام:

    • جافا سكريبت يمكن استخدامها لإظهار/إخفاء عناصر الصفحة، إنشاء القوائم المنسدلة، التحقق من صحة النماذج، إضافة تأثيرات بصرية، إلخ.


  • تطبيقات أخرى:

    • جافا سكريبت لم تعد مقتصرة على المتصفحات. ظهرت Node.js وهي بيئة عمل تسمح بتنفيذ جافا سكريبت على الخوادم، مما يوسع استخدامها لتشمل تطوير الواجهة الخلفية (Back-end) وتطبيقات أخرى.


    بشكل عام، جافا سكريبت هي لغة برمجة قوية ومستخدمة على نطاق واسع، وهي أساسية لتطوير الويب التفاعلي الحديث.
